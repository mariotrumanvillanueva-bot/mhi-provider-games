# MHI Friday Mini Games – Provider Coordinators

GitHub Pages site for the controlled Provider Coordinators game.

## Files included
- index.html
- style.css
- config.js
- gameBank.js
- game.js

## Before publishing
Open `config.js` and replace:

```js
const API_URL = "PASTE_BACKEND_WEB_APP_URL_HERE";
```

with your Google Apps Script Web App URL.

Example:

```js
const API_URL = "https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec";
```

## Suggested GitHub repository name
mhi-provider-games

## GitHub Pages URL
After setup, your site will look like:

https://YOURUSERNAME.github.io/mhi-provider-games/
